This CDN import fails in Node. Solution: Use `vi.mock` to mock `../../scripts/core/Utils.js` partially, replacing `loadFuse` with a function that returns the real Fuse.js constructor from the devDependency:
   ```javascript
   import Fuse from 'fuse.js';
   vi.mock('../../scripts/core/Utils.js', async (importOriginal) => {
     const actual = await importOriginal();
     return { ...actual, loadFuse: vi.fn(async () => Fuse) };
   });
   ```
   This lets all other Utils exports (isExcludedPath, getCreatureCacheKey, etc.) work normally.
